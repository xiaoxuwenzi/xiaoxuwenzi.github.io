分类列表

此页面用于展示文章分类。Jekyll 的归档插件 (jekyll-archives) 已在 `_config.yml` 中配置，可以自动生成按年份和标签归档的页面。如果需要按自定义分类展示，可以进一步配置或手动创建链接。

目前的 `_config.yml` 配置了 `jekyll-archives` 来处理 `year` 和 `tags` 归档，访问路径分别为 `{{ site.baseurl }}/archives/:year/` 和 `{{ site.baseurl }}/tags/:tag/`。

您可以考虑使用标签云或更详细的分类列表来丰富此页面。以下是如何手动列出所有标签的示例：

所有标签

{% for tag in site.tags %} #{{ tag[0] }} ({{ tag[1].size }}) {% endfor %}

所有分类

{% for category in site.categories %} {{ category[0] }} ({{ category[1].size }}) {% endfor %}
