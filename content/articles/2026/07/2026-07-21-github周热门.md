本周 GitHub Trending 榜单上，AI Agent 相关项目几乎全面霸榜。Nutlope/hallmark（8.8k Star）是一款反 AI 味设计技能，内置 20 套主题与 57 道 slop-test 门控，让 Claude Code、Cursor、Codex 生成的 UI 不再千篇一律；OpenCut 则是开源的 CapCut 替代方案，正以 Rust 核心重写，引入插件架构、MCP 服务器和无头批渲染。

Shubhamsaboo/awesome-llm-apps 收录了 100+ 可直接运行的 AI Agent 与 RAG 应用，覆盖语音、研究、金融、医疗多场景，支持一键安装为编码 Agent 的新技能；HKUDS/DeepTutor 把辅导、解题、测验、研究、可视化整合在同一个 Agent 循环上，支持多引擎知识库和可检视的三层记忆。

> AI Agent 正从「单点工具」走向「生态平台」——Agent 能力像 npm 包一样被模块化、可发现、可组合。

Agent 安全是本周涌现的新独立赛道。Dicklesworthstone/destructive_command_guard（5.2k Star，本周新增 1.4k）用 Rust 实现亚毫秒级危险命令拦截，支持 10+ Agent 和 50+ 安全包；tirth8205/code-review-graph 用 Tree-sitter 构建代码图谱，在 6 个真实仓库基准测试中 Token 消耗中位数降低约 82 倍，让 AI 只读真正必要的文件。

垂直领域 Agent 深入行业流程。iOfficeAI/OfficeCLI 是首个专为 AI Agent 设计的 Office 套件，单二进制即可读写编辑 Word/Excel/PowerPoint，内置 HTML 渲染引擎让 AI「看见」自己的输出并闭环修正；tt-a1i/archify（6.25k Star，本周新增 2.1k）则用自然语言生成架构、时序、数据流图，支持暗亮主题与多格式导出。

终端编码 Agent 竞争白热化。openai/codex 以 Rust 重写，本周合并 30+ PR 持续迭代；earendil-works/pi（55.6k Star）将多供应商 LLM API、Agent 运行时与编码 CLI 拆分为可组合的包，并采用严格的供应链加固。HKUDS/Vibe-Trading 则把量化研究的完整链路封装为 Agent 技能，集成 460+ 学术因子与 16 适配器 IM 通道。
