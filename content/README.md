# content/ 内容目录

存放所有资讯内容的核心目录。

## 子目录说明

| 目录 | 用途 | 说明 |
|------|------|------|
| `articles/` | 已发布资讯 | 按 `年/月/` 归档,如 `2026/07/2026-07-21-标题.md` |
| `drafts/` | 草稿 | 写好但未发布的内容,发布后移到 articles/ |
| `sources/` | 原始素材 | 收集的资料、链接、笔记,按分类存放 |
| `assets/` | 文章配图 | 文章中用到的图片资源 |

## 文件命名规范

已发布文章:`YYYY-MM-DD-标题简写.md`

示例:`2026-07-21-大模型成本下降.md`

## 工作流程

```
1. 收集素材 → content/sources/AI/收集笔记.md
2. 写草稿   → content/drafts/新文章.md
3. 发布     → content/articles/2026/07/2026-07-21-标题.md
4. 更新网站 → 编辑 index.html 的 newsData 数组
```
